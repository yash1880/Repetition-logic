Smart Café Billing System
A simple web-based billing system for a café that calculates the bill amount with discounts and GST for selected items.

Overview
This project implements a straightforward café billing system using HTML, CSS, and JavaScript. The system allows users to enter quantities for Coffee, Burger, and Pizza, applies a 10% discount on each item, calculates GST at 5%, and generates a detailed bill.

Features
Input fields to enter quantity for each item.

Each item price is fixed (Coffee ₹150, Burger ₹200, Pizza ₹410).

10% discount automatically applied on each item’s original bill.

Calculates subtotal after discount.

Adds a 5% GST tax on the subtotal.

Displays detailed bill including original bill, discount, final bill per item, subtotal, GST, and grand total.

Clean and modern UI with responsive table and styled buttons.

Usage
Clone or download the repository.

Open index.html (or the main HTML file) in a web browser.

Enter the quantity for Coffee, Burger, and Pizza.

Click the Generate Bill button.

The detailed bill will be displayed below the button.

Code Highlights
HTML5 for structure and semantic markup.

CSS for styling the layout and elements.

JavaScript handles form input, calculations for discount and GST, and dynamically updates the bill details.
